# dotfiles
Simple macOS dotfiles and exported application settings.

# Installation
1. Install Homebrew - `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
2. Run `install.sh`

# Restoring exported settings
1. Open iTerm2 → Preferences → General → Settings.
2. Enable "Load preferences from a custom folder or URL".
3. Point it to the `exported_settings` folder in this repository and enable automatic loading.
